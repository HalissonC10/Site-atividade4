# Landing Page

## Descrição
Este projeto consiste em uma landing page moderna, desenvolvida com HTML, CSS, JavaScript e bibliotecas externas, consumindo APIs para exibir serviços e depoimentos. 

## Como Visualizar o Projeto

1. Clone o repositório para sua máquina local:
https://github.com/HalissonC10/Site-atividade4.git

2. Navegue até o diretório do projeto:

3. Abra o arquivo `index.html` em seu navegador.

## Estrutura do Projeto

- `index.html`: O arquivo HTML principal.
- `styles.css`: Estilos para a página.
- `script.js`: Lógica JavaScript para a interatividade da página.
  
- APIs utilizadas:
- **RandomUser.me** para carregar depoimentos dinâmicos.
- **EmailJS** para o envio de formulários.

## Bibliotecas Externas

- **Swiper.js**: Utilizada para criar um slider/carrossel para o banner e serviços.
- **AOS.js**: Para animações ao rolar a página.
- **EmailJS**: Para enviar mensagens de contato diretamente do formulário.

## Requisitos

- Um navegador moderno (Chrome, Firefox, etc.) para visualização da página.
